# 🤖 AI Task Manager 🚀

A modern, interactive **Task Management Web App** built with **Streamlit**, featuring authentication, smart task tracking, and a visually appealing UI.

---

## 📌 Overview

AI Task Manager is a productivity-focused application that allows users to:

* 🔐 Securely create accounts and log in
* 📝 Manage daily tasks efficiently
* 🎯 Set priorities and deadlines
* ✅ Track completion status
* ⚠️ Get intelligent task suggestions

Designed with a **glassmorphism UI** and dynamic particle background, this app delivers both functionality and aesthetics.

---

## ✨ Features

### 🔐 Authentication System

* User signup and login
* Password hashing using SHA-256
* SQLite database integration

### 📋 Task Management

* Add tasks with:

  * Priority (High 🔥 / Medium 🤔 / Low 📉)
  * Due date 📅
* Mark tasks as completed ✔️
* Delete tasks 🗑️

### 🧠 AI Suggestions

* Detect overdue tasks ⚠️
* Highlight high-priority items 🔥
* Provide productivity insights

### 🎨 UI/UX Design

* 🌙 Dark/Light mode toggle
* ✨ Glassmorphism cards
* 🌌 Animated particle background
* Responsive layout using Streamlit

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/hemant467/Task-Manager.git
cd Task-Manager
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the app

```bash
streamlit run Taskmanager4.py
```
